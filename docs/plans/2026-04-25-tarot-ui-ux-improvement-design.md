# 타로 UI/UX 개선 디자인 문서

## 1. 개요
모바일 환경에서 78장의 카드를 선택할 때 발생하는 스크롤 피로도를 줄이고, 선택한 카드를 확인하는 과정에서 역동적인 연출을 추가하여 사용자 경험을 고도화함.

## 2. 주요 변경 사항

### 2.1 가로 중첩 슬라이드 스프레드 (Picking UI)
- **구조**: `overflow-x-auto`를 사용한 가로 긴 줄 배치.
- **스타일**: 카드가 서로 살짝 겹쳐 보이도록 음수 마진 적용.
- **인터랙션**: 모바일 스와이프를 통한 카드 탐색, 선택 시 시각적 피드백 강화.

### 2.2 순차적 카드 공개 시퀀스 (Revelation Phase)
- **신규 단계**: `tarot_revealing` 단계 추가.
- **연출**: 선택된 3장의 카드가 중앙에서 크게 확대되며 순차적으로 뒤집힘.
- **타이밍**: 각 카드 공개 사이의 대기 시간을 1초로 설정하여 빠른 템포 유지.
- **흐름**: 3장 선택 완료 -> 순차 공개 -> 최종 리포트 자동 전환.

## 3. 기술적 세부 사항
- **상태 관리**: `step` 타입에 `'tarot_revealing'` 추가, 현재 공개 중인 카드 인덱스를 추적하는 `revealingIndex` 상태 추가.
- **애니메이션**: `framer-motion`의 `AnimatePresence`와 `LayoutGroup`을 활용하여 부드러운 전환 구현.
- **컴포넌트**: `MinhwaCard` 컴포넌트의 확대/뒤집기 모드 대응.

## 4. 기대 효과
- 모바일 가독성 및 조작 편의성 향상.
- 운명적인 선택에 대한 몰입감 및 재미 요소 증대.
